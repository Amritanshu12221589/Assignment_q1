# Q1 — Student Registration Screen (Jetpack Compose)

## CSE 225: Developing Android Apps — CA1

### Features
- Jetpack Compose UI with OutlinedTextField, RadioButton, Checkbox, Button
- Real-time form validation (Name, Roll No, Email, Gender, T&C)
- Success result card displayed on valid submission

### How to Run
1. Open Android Studio
2. File → Open → select this folder
3. Wait for Gradle sync
4. Run ▶ on emulator/device (API 24+)

### Package
`com.example.studentregistration`
